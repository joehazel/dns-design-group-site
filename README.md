# DNS Design Group Website

This is a ready-to-upload Next.js project for the DNS Design Group website.

## What this is
- A simple website project you can upload to GitHub
- Deployable on Vercel
- Built with Next.js and Tailwind CSS

## How to use it
1. Unzip the folder
2. Upload all files to a new GitHub repository
3. Import that repository into Vercel
4. Click Deploy

## Important note
The images are currently placeholder images from Unsplash. Replace them later with your real project photos.

## Local development (optional)
If someone technical wants to run it locally:

```bash
npm install
npm run dev
```
